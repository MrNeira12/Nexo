# Nexo
🎵 Nexo: Academic Music & Media Player
A specialized Flutter-based multimedia application for students and educators.

📝 Description
EduStream (o el nombre que prefieras) is a modern, cross-platform mobile application designed to bridge the gap between education and entertainment. Inspired by the sleek interface of top-tier music streaming services, this app allows users to manage and consume academic content in both audio (MP3) and video formats.

Whether it’s a recorded lecture stored in Firebase or an educational tutorial from YouTube, EduStream provides a unified, immersive experience with a focus on usability and aesthetic design.

🚀 Key Features
Dual-Media Playback: Seamlessly switch between streaming local/cloud audio files and YouTube educational videos.

Firebase Integration: Robust backend support using Firebase Authentication, Firestore for metadata management, and Firebase Storage for hosting high-quality audio files.

Modern UI/UX: A clean, "Spotify-inspired" interface featuring dynamic glassmorphism effects, blur backgrounds, and smooth transitions.

Smart Media Controller: A custom-built player that synchronizes progress bars, play/pause states, and metadata across different media engines.

Academic Library: An organized repository where students can browse content by title, author, or category.

🛠️ Technical Stack
Framework: Flutter (Dart)

Backend: Google Firebase (Firestore, Storage, Auth)

Media Engines: Youtubeer_flutter & audioplayers

State Management: StatefulWidgets with optimized lifecycle handling (dispose/init).
